실행파일 : run.py

필요 라이브러리 설치  : 
명령어(cmd or 가상환경) 작성 _ $pip install -r requirements.txt

패키지 사용법 : 
run.py 디버깅(실행)
	|
출력 내용 : 
[로컬 접속 :  * Running on http://127.0.0.1:8801]
[외부 접속 :  * Running on http://221.123.456.84:8801]
-> 외부 접속 IP 주소를 통해 웹앱 접근
	|
Location Guide 앱 동작

사용자 기능 : 
메인페이지 -> 로그인 -> 회원가입 -> 로그인 -> 지도선택 -> 특정 위치 검색 or 지정 위치 마커 클릭 -> 정보출력
(지도선택 ~ 정보출력 내부 좌측 상단 아이콘 클릭 시, 지도선택 화면으로 이동)
(정보출력 좌측 상단 뒤로가기 아이콘 클릭 시, 이전 지도 화면으로 이동)
